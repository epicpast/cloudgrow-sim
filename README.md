# cloudgrow-sim

A simulator for greenhouse environments to facilitate training and estimations of tooling and servicing.

## Installation

```bash
# Using uv (recommended)
uv add cloudgrow-sim

# Using pip
pip install cloudgrow-sim
```

## Development

```bash
# Clone the repository
git clone https://github.com/OWNER/cloudgrow-sim.git
cd cloudgrow-sim

# Install with dev dependencies
uv sync

# Run tests
make test

# Run all quality checks
make quality
```

## Usage

```python
from cloudgrow_sim import main

# Example usage
main()
```
